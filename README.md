# Movie Recommendation System

This project implements a movie recommendation system using real-world data from the MovieLens dataset.

## Overview
The system recommends movies based on similarity between user rating patterns. It uses collaborative filtering techniques to identify similar movies.

## Features
- Real dataset (MovieLens)
- Data preprocessing and merging
- User-movie matrix creation
- Cosine similarity for recommendations

## Technologies
- Python
- Pandas
- Scikit-learn

## Example
Input: Toy Story (1995)  
Output: List of similar movies based on user ratings
